## pegadinhas

### Registro (120–129-B)

| Art. | Regra-chave |
|------|-------------|
| **120** | Registro obrigatório: **automotor, articulado, reboque, semirreboque** — órgão executivo de trânsito do **Estado ou DF**, município de domicílio/residência do proprietário |
| **120 §1º** | Veículos **oficiais** (administração **direta** União/Estados/DF/Municípios): pintura nas portas com nome/sigla/logotipo — **exceto** representação diplomática e art. **116** (policial reservado) |
| **120 §2º** | **Não** se aplica a veículo de **uso bélico** — ônibus/caminhão do Exército **não** são bélicos por si só |
| **121** | CRV (Certificado de Registro de Veículo) — físico e/ou digital; **sem prazo de validade**; modelo CONTRAN |
| **122** | Expedição CRV: consulta **RENAVAM** + nota fiscal (I) ou documento **MRE** para importado diplomático (II) |
| **123** | Novo CRV obrigatório: (I) transferência propriedade; (II) mudança **município** domicílio/residência; (III) alteração característica; (IV) mudança categoria |
| **123 §1º** | Transferência: prazo **30 dias**; demais casos: providências **imediatas** |
| **123 §2º** | Mudança endereço **no mesmo município**: comunicar em **30 dias**; alterar CRLV só no **novo licenciamento** |
| **124** | Documentos para novo CRV (I–XI): CRV anterior, CRLV, comprovante transferência, certificado segurança/poluentes se adaptado, certidão roubo/furto (ou RENAVAM), **quitação débitos** (VIII), etc. |
| **124 parágrafo único** | Transferência por **apreensão, confisco, leilão ou doação** a ente público: **dispensado** inciso VIII (débitos) |
| **125** | Dados ao RENAVAM: fabricante/montadora (nacional); **alfândega** (importado **PF**); **importador** (importado **PJ**) |
| **126** | Baixa: veículo irrecuperável ou desmontagem — CONTRAN; **vedada remontagem** no mesmo chassi; obrigação pode ser da **seguradora** ou adquirente para desmonte |
| **126 §2º** | Débitos fiscais/multas **não impedem** a baixa |
| **127** | Baixa só após consulta RENAVAM; comunicar ao RENAVAM **de imediato** |
| **128** | **Não** expede novo CRV com débitos fiscais/multas trânsito/ambientais vinculados ao veículo — **independente** de quem cometeu a infração |
| **129** | Propulsão humana e tração animal: registro/licenciamento por **legislação municipal** |
| **129-A** | Tratores e aparelhos agrícolas: registro **sem ônus** pelo **Ministério da Agricultura** (ou convênio) |
| **129-B** | Contratos de garantia (alienação fiduciária, consórcio, arrendamento, reserva domínio, penhor): registro no DETRAN; empresas registradoras credenciadas |

### Licenciamento (130–135)

| Art. | Regra-chave |
|------|-------------|
| **130** | Licenciamento **anual** — órgão executivo do **Estado/DF** onde o veículo está registrado; exceção: uso **bélico** |
| **130 §2º** | Mudança domicílio/residência no **mesmo exercício**: vale licenciamento de **origem** (novo só no ano seguinte) |
| **131** | CRLV vinculado ao CRV; 1º licenciamento **simultâneo** ao registro |
| **131 §2º** | Só licenciado com tributos, encargos e multas trânsito/**ambientais** quitados — vinculados ao veículo (ADI 2998) |
| **131 §3º** | Licenciar exige aprovação nas inspeções do art. **104** |
| **131 §4º** | Informações de **recall** desde 01/10/2019 no licenciamento; prorrogação atendimento recall até **1 ano** se falta peça/escalonamento |
| **132** | Veículos **novos** isentos de licenciamento no trajeto fábrica/alfândega → município destino (CONTRAN regula) |
| **133** | Porte do CRLV **obrigatório** — dispensado se fiscalização acessar sistema informatizado |
| **134** | Se comprador não registra em **30 dias** (art. 123 §1º), **antigo proprietário** envia ao DETRAN, em **60 dias**, cópia autenticada do comprovante de transferência — senão **responsabilidade solidária** por multas/reincidências até comunicação |
| **134-A** | CONTRAN especifica bicicletas motorizadas **não** sujeitas a registro, licenciamento e emplacamento |
| **135** | Veículo de **aluguel** (táxi, ônibus remunerado etc.): registro/licenciamento/emplacamento comercial só com autorização do **poder concedente** (município/estado conforme serviço) |

**Reboque/semirreboque:** não pagam IPVA/DPVAT; pagam licenciamento anual e recebem CRLV.

### Condução de escolares (136–139)

| Art. | Regra-chave |
|------|-------------|
| **136** | Só circula com **autorização** do órgão executivo de trânsito dos **Estados e DF** (não União) |
| **136 exigências** | I registro **passageiros**; II inspeção **semestral**; III faixa **amarela 40 cm** à meia altura + dístico **ESCOLAR** preto (cores invertidas se carroceria já amarela); IV **tacógrafo**; V lanternas topo dianteira/traseira; VI cintos = lotação; VII demais CONTRAN |
| **137** | Autorização **afixada** no interior, visível, com lotação permitida; **vedado** escolares acima da capacidade do fabricante — **sem** crianças em pé |
| **138** | Condutor: I idade **>21**; II CNH categoria **D**; III **(VETADO)**; IV **máx. 1 gravíssima** nos últimos **12 meses**; V curso especializado CONTRAN |
| **139** | Capítulo **não exclui** competência **municipal** para exigências adicionais |

- Transporte escolar **não** exige cadeirinha/bebê-conforto (diferente de automóvel particular).

### Moto-frete (139-A, 139-B)

**Atenção MP 1.360/2026:** caput revogou exigência de **autorização** estadual/DF e incisos **I** (registro aluguel) e **IV** (inspeção semestral). Texto vigente:

| Dispositivo | Conteúdo atual |
|-------------|----------------|
| **139-A caput** | Moto-frete só circula com os requisitos dos incisos II e III |
| **II** | Protetor de motor **mata-cachorro** no chassi (CONTRAN) |
| **III** | **Corta-pipas** (CONTRAN) |
| **§1º** | Dispositivo de carga conforme CONTRAN |
| **§2º** | **Proibido** combustíveis, inflamáveis, tóxicos e galões — **exceto** gás de cozinha e galões de água mineral com **side-car** |
| **139-B** | Não exclui competência **municipal ou estadual** para regulamentar moto-frete na circunscrição |

- Questões da apostila que citam autorização do DETRAN estadual ou registro **aluguel** refletem redação **pré-MP 1.360** — validar literalidade na prova.

## nao_confundir_com

| Confusão | Correção |
|----------|----------|
| Registro no município | No **DETRAN do Estado/DF** do domicílio do proprietário |
| Veículo militar = bélico | Só **uso bélico** (art. 120 §2º) |
| CRV com validade anual | CRV **sem validade**; anual é o **CRLV** (licenciamento) |
| Multa do empréstimo do carro | Débitos vinculam o **veículo** — bloqueiam CRV (128) e licenciamento (131 §2º) |
| Prazo 30 dias vs 60 dias | **30 dias** = comprador registrar (123 §1º); **60 dias** = vendedor comunicar DETRAN se comprador não registrou (134) |
| RENAVAM importado | **PF** → alfândega; **PJ** → importador |
| Escolar: autorização União | **Estados e DF** (136) |
| Escolar: categoria B ou E | Categoria **D** (138 II) |
| Escolar: cadeirinha obrigatória | **Não** está no art. 136 |
| Moto-frete: autorização SENATRAN | Apostila antiga: **Estados/DF**; MP 1.360/2026: caput **sem** autorização no texto atual |
| Moto-frete = mototáxi | Capítulos e requisitos **diferentes** |
| Art. 139 vs 139-B escolar/moto | **139** = escolar + competência municipal; **139-B** = moto-frete + municipal **ou** estadual |
