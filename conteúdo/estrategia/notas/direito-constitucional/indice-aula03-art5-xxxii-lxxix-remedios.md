## 1) art. 5º — incisos XXXII a LXXIX (mapa)

### Administrativos e acesso

| Inc. | Conteúdo | Pegadinha |
|------|----------|-----------|
| **XXXII** | Defesa do consumidor (eficácia **limitada**; CDC 8.078/90) | Bancos = CDC; **aviação internacional** = Convenções Varsóvia/Montreal prevalecem (RE 636.331) |
| **XXXIII** | Direito à informação (órgãos públicos + indiretas) | Sigilo imprescindível à segurança sociedade/Estado; base da LAI |
| **XXXIV** | Petição + certidões **sem taxas** | Petição: escrita, sem formalismo; interesse próprio, coletivo ou de terceiros |
| **XXXV** | Inafastabilidade da jurisdição | Sistema **inglês** (jurisdição una); decisão administrativa sempre controlável judicialmente |
| **XXXVI** | Segurança jurídica (direito adquirido, AJ perfeito, coisa julgada) | — |
| **XXXVII** | Vedação a juízo/tribunal de **exceção** | Com LIII = **juiz natural** |
| **XXXVIII** | Júri | — |
| **XXXIX** | Legalidade penal (nullum crimen) | Com XL = **irretroatividade** penal |

### Direito penal e extradição

| Inc. | Conteúdo |
|------|----------|
| **XL** | Lei penal **não retroage**, salvo para **beneficiar** o réu |
| **XLI** | Lei pune discriminação atentatória a direitos fundamentais |
| **XLII** | Racismo = crime **inafiançável** e **imprescritível** |
| **XLIII** | Tortura, tráfico, terrorismo = inafiançáveis e sem graça/anistia |
| **XLIV** | Milícias/grupos armados = inafiançável e imprescritível |
| **XLV–L** | Pena não passa da pessoa; individualização; penas vedadas; regime prisional; presidiárias + filhos na amamentação |
| **LI** | **Nenhum brasileiro** extraditado, salvo **naturalizado** (crime comum antes da naturalização ou tráfico) |
| **LII** | Extradição de estrangeiro vedada por crime **político** ou de **opinião** |

### Devido processo e prisão (9.3 — alta relevância STTP)

| Inc. | Conteúdo |
|------|----------|
| **LIII** | Processo/sentença só por autoridade **competente** (juiz natural) |
| **LIV** | Devido processo legal — formal (contraditório, ampla defesa, provas lícitas) + material (proporcionalidade) |
| **LV** | Contraditório e ampla defesa a todos |
| **LVI** | Provas **ilícitas** inadmissíveis |
| **LVII** | **Presunção de inocência** até trânsito em julgado |
| **LVIII** | Identificação civil ≠ criminal (salvo hipóteses legais) |
| **LIX–LX** | Ação penal privada; publicidade processual |
| **LXI–LXVI** | Prisão: flagrante ou ordem escrita/fundamentada de autoridade competente; comunicação; direitos do preso (LXIII: calado, advogado, família); relaxamento |
| **LXVII** | **Prisão civil** só por alimentos (voluntário + inescusável) — STF: **única** admitida (depositário infiel **não**) |

### Remédios e garantias finais (texto constitucional)

| Inc. | Remédio / direito |
|------|-------------------|
| **LXVIII** | Habeas corpus |
| **LXIX** | Mandado de segurança individual |
| **LXX** | Mandado de segurança coletivo |
| **LXXI** | Mandado de injunção |
| **LXXII** | Habeas data |
| **LXXIII** | Ação popular |
| **LXXIV** | Assistência jurídica **integral e gratuita** (comprovar insuficiência) |
| **LXXV** | Indenização por **erro judiciário** |
| **LXXVI** | Gratuidade de atos para pobres (certidões, RC) |
| **LXXVII** | HC e HD **gratuitos**; AP isenta custas e sucumbência (salvo má-fé) |
| **LXXVIII** | Duração razoável do processo + meios célere |
| **LXXIX** | Proteção de **dados pessoais** (LGPD) |

## 2) habeas corpus (art. 5º, LXVIII)

### Conceito

Protege **liberdade de locomoção** contra violência/coação por ilegalidade ou abuso de poder.

| Aspecto | Regra |
|---------|-------|
| Tipos | **Repressivo** (já preso) / **Preventivo** (ameaça) / Suspensivo |
| Legitimado **ativo** | Universal — PF, PJ, MP, Defensoria; **sem advogado** |
| **Paciente** | Só **pessoa física** (PJ pode impetrar, não ser paciente) |
| Passivo | Autoridade pública **ou particular** |
| Natureza | **Penal**; rito sumário; **gratuito** (LXXVII) |
| Liminar | Sim (fumus + periculum) |
| Ofício | Juiz pode conceder de ofício |
| Coletivo | STF admite (ex.: grávidas/mães em prisão preventiva) |

### Cabimento

- Medidas cautelares diversas da prisão (art. 319 CPP)  
- Quebra sigilo bancário/fiscal em **processo penal** (não em administrativo fiscal)  
- Medidas Maria da Penha que restrinjam locomoção  
- Pena pecuniária **conversível** em privativa de liberdade  

### Incabível

- Contra decisão do **STF** (Plenário, Turma, ministro) — Súm. **606**  
- Pena de **multa** isolada (Súm. **693**)  
- Suspensão direitos políticos; sanção **disciplinar** administrativa  
- Quebra sigilo sem risco de prisão  
- Pena já **extinta** (Súm. **695**)  
- Visitas íntimas; custas do HC (Súm. **395**)  
- Estado defesa/sítio: pode **restringir**, nunca **suprimir**

## 3) habeas data (art. 5º, LXXII)

| Aspecto | Regra |
|---------|-------|
| Finalidade | (a) **Acesso** a dados sobre o impetrante em registros/bancos públicos; (b) **Retificação** |
| Legitimado | PF ou PJ — ação **personalíssima** (não dados de terceiros) |
| Exceção | Cônjuge sobrevivente (interesse do falecido) |
| Passivo | Entes públicos ou privados com banco de dados de **caráter público** |
| Natureza | **Civil**; gratuito; **exige advogado** |
| Liminar | **Não** |
| Condição | **Negativa prévia** da administração (interesse de agir) |
| Decadência | **Não** |
| Prioridade | Após HC e MS |
| Lei | 9.507/97 |

**Não serve** para vista de processo administrativo → use **MS**.

## 4) mandado de segurança individual (art. 5º, LXIX)

| Aspecto | Regra |
|---------|-------|
| Finalidade | Direito **líquido e certo** não amparado por HC/HD |
| Caráter | **Residual** |
| Passivo | Autoridade pública ou particular no exercício de função pública |
| Natureza | **Civil** (pode ser usado em processo penal) |
| Prova | **Pré-constituída** (sem dilação probatória) |
| Prazo | **120 dias** da ciência oficial do ato — **decadencial** (Súm. 632) |
| Liminar | Sim (ADI 4296: vedou §2º art. 7º Lei 12.016 — liminar sempre possível) |
| Honorários | **Não** há sucumbência |
| Reexame | Obrigatório se **concedida** a segurança |
| Lei | 12.016/2009 |

**Direito líquido e certo** = evidente de plano; controvérsia jurídica **não impede** (Súm. 625).

### Incabível

- Decisão judicial com recurso **suspensivo**  
- Lei **em tese** (salvo efeitos concretos)  
- Decisão **transitada em julgado**  
- Sucedâneo recursal  
- Contra STF (salvo exceção)  
- Contra MP editando tributo por MP  

**Ilegalidade** → ato vinculado; **abuso de poder** → ato discricionário.

---

## 5) mandado de segurança coletivo (art. 5º, LXX)

| Legitimado ativo | Requisito |
|------------------|-----------|
| Partido político | Representação no **Congresso Nacional** |
| Sindicato / entidade de classe | Sem prazo mínimo |
| Associação | Constituída e funcionando há **≥ 1 ano** |

- Direitos **coletivos** ou individuais **homogêneos** — **não difusos** (use ACP)  
- Substituição processual — **sem** autorização dos associados (Tema 1.119)  
- Entes federativos **não** podem impetrar  
- Partido **não** pode MS coletivo para impugnar majoração de tributo

## 6) mandado de injunção (art. 5º, LXXI)

| Aspecto | Regra |
|---------|-------|
| Finalidade | Suprir **omissão** de regulamentação que inviabilize direito/liberdade/prerrogativa (nacionalidade, soberania, cidadania) |
| Cabimento | Normas de eficácia **limitada** de caráter **impositivo** |
| Legitimado | Qualquer PF ou PJ |
| Gratuidade | **Não** — exige advogado |
| Liminar | **Não** (STF) |
| Lei | 13.300/2016 |
| Eficácia decisão | **Concretista individual** (art. 9º) — até lei regulamentadora; pode erga omnes em casos excepcionais |

### Pressupostos

1. Falta de norma regulamentadora (programática impositiva)  
2. Nexo causal omissão → impossibilidade de exercer direito  
3. Prazo razoável decorrido (mora legislativa)

### Incabível

- Já existe regulamentação (mesmo defeituosa)  
- Omissão de lei **infraconstitucional**  
- MP não convertida em lei  
- Regulamentação **facultativa** (não obrigatória)

**Coletivo:** partido (CN), sindicato/classe/associação (1 ano), MP, Defensoria.

**Competência:** ratione personae — STF ou STJ conforme autoridade omissa.

Ex.: SV 33 (aposentadoria especial servidor); greve servidor → Lei 7.783 temporária.

## 7) ação popular (art. 5º, LXXIII)

| Aspecto | Regra |
|---------|-------|
| Finalidade | Anular ato lesivo a patrimônio público, moralidade administrativa, meio ambiente, patrimônio histórico/cultural |
| Legitimado | Só **cidadão** (PF com direitos políticos — eleitor) — **não** PJ, partido, MP originário |
| Custas/sucumbência | Isento (salvo **má-fé** comprovada) |
| Dano material | **Não** precisa comprovar prejuízo (Tema 836) |
| Preventiva ou repressiva | Sim |
| Contra decisão judicial | **Não** (ato administrativo) |
| Foro privilegiado | **Não** há |
| Improcedência | Reexame necessário |

MP: custos legis, auxiliar, substituto (autor omisso), sucessor (se autor desiste); **não** autor originário (prevalece STF).

---

## 8) ação civil pública (art. 129, III — não é remédio do art. 5º)

| Aspecto | Regra |
|---------|-------|
| Base | CF art. 129 III + Lei **7.347/85** (+ CDC para individuais homogêneos) |
| Objeto | Interesses **difusos**, **coletivos**, individuais **homogêneos** |
| Legitimados (rol **taxativo**) | MP, Defensoria, União, Estados, DF, Municípios, autarquias/EP/SEM, associação (≥1 ano + finalidade na lei) |
| MP | **Não** exclusivo; fiscal da lei se não for parte |
| Tributos | MP **não** pode ACP para cobrança de tributos |
| Mensalidade escolar | MP **pode** (Súm. 643) |

## pegadinhas

| Erro | Correção |
|------|----------|
| PJ é paciente de HC | Só **PF** é paciente |
| HC só após prisão concretizada | **Preventivo** cabe |
| HC contra ministro STF | **Não** cabe (Súm. 606) |
| HD para processo administrativo | **MS** |
| HD sem negativa prévia | Exige **pretensão resistida** |
| MS para direito difuso | **ACP** ou MS coletivo (homogêneo) |
| Qualquer partido = MS coletivo | Com representação no **CN** |
| 1 ano para sindicato MS coletivo | Só **associação** |
| MI para omissão de lei ordinária | Só omissão de norma **constitucional** |
| MI com liminar | **Não** |
| AP por PJ ou MP | Só **cidadão** |
| AP gera sucumbência | Isento salvo **má-fé** |
| ACP exclusiva do MP | Rol **plural** de legitimados |
| Pena de morte vedada sempre | Só em **guerra declarada** (inc. XLII questões: tortura nunca) |
| Prisão civil por qualquer dívida | Só **alimentos** (STF) |
| Brasileiro nato extraditado | **Não** (salvo naturalizado nos casos do LI) |
| CDC em voo internacional | **Varsóvia/Montreal** |
| Reunião = HC | **MS** (aula 02) |
| Depósito infiel = prisão civil | STF **vedou** — só alimentos |
