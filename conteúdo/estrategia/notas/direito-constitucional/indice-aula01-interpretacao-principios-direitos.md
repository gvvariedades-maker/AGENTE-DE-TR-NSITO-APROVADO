## 1) aplicação das normas constitucionais no tempo

### Efeitos da nova Constituição (três eixos)

| Efeito | Regra brasileira |
|--------|------------------|
| **a) Constituição anterior** | **Revogada integralmente** — sai do mundo jurídico |
| **b) Leis compatíveis** | **Recepcionadas** (tácita, automática) |
| **c) Leis incompatíveis** | **Revogadas** (tácita, automática) |

### Desconstitucionalização

- Teoria estrangeira: nova Carta **recepciona** normas da anterior como infraconstitucionais.  
- **Brasil não adota** — nem genericamente nem por dispositivo. Só ocorreria com **determinação expressa** do PCO.  
- **Pegadinha:** já foi cobrada em prova, mas **não vale** no Brasil.

### Recepção — requisitos

| Critério | Exige? |
|----------|--------|
| Compatibilidade **material** (conteúdo) | **Sim** |
| Compatibilidade **formal** (espécie normativa) | **Não** |
| Norma **válida** na Constituição da época | **Sim** |
| Norma **em vigor** na promulgação | **Sim** (vacatio legis = **não** recepciona) |

**Exemplo CTN:** editado como LO sob CF/46 → recepcionado pela CF/67 **como LC** (status definido pela nova Carta). CF/88 manteve LC.

- Recepção pode ser **parcial** (artigo por artigo).  
- CF/88 pode **determinar expressamente** continuidade de dispositivos da CF anterior (ex.: sistema tributário até 5º mês).  
- **Repristinação:** só **expressa**; em regra só recepciona o que **estava em vigor** na promulgação.

### Incompatíveis = revogação, não inconstitucionalidade superveniente

- **STF:** controle de constitucionalidade só para norma **contemporânea** à CF vigente na edição.  
- Lei de 1982 (CF/67) **não** é julgada pela CF/88 → conflito no tempo = **revogação**.  
- **Exceções (Lenza):** (i) **mutação constitucional**; (ii) mudança do **substrato fático** (ex.: amianto ADI 3.937).

### Emenda constitucional

- Lei anterior **incompatível** com EC → **revogada** (mesma lógica da recepção).  
- Lei **posterior** à EC e incompatível → pode ser declarada **inconstitucional**.

### Casos especiais

| Situação | Resultado |
|----------|-----------|
| Lei **inválida** na CF pretérita, mas compatível com a nova | **Não** recepciona (nasceu inválida) |
| Mudança de **competência** (maior → menor grau) | Lei federal pode virar estadual |
| Mudança (menor → maior) | 27 leis estaduais **não** viram federais |
| Recepção precisa ser **expressa**? | **Não** — basta compatibilidade material |

## 2) interpretação constitucional

### Quem interpreta?

- **Judiciário, Executivo e Legislativo** — não é exclusividade do Judiciário.  
- **Häberle:** “sociedade aberta dos intérpretes” (cidadãos, órgãos, opinião pública).  
- **STF:** última palavra quando instado.

### Correntes sobre o juiz

| Corrente | Posição |
|----------|---------|
| **Interpretativistas** | Limitar-se ao texto + implícitos claros |
| **Não interpretativistas** | Valores substantivos (justiça, liberdade, igualdade); “Constituição aberta” (Canotilho) |

**Pegadinha:** “não interpretacionismo = limitar ao texto” → **errado** (isso é interpretativismo).

### Mutação constitucional

- Alteração **informal** — texto **intacto**, muda o **sentido**.  
- Manifestação do **poder constituinte difuso** (derivado, não escrito).  
- STF (ADI 5540): legitima quando há (a) mudança na percepção do direito; (b) modificação fática; (c) consequência negativa de entendimento.  
- Constituições **rígidas** → mais mutações.

### Métodos principais

| Método | Autor / ideia |
|--------|----------------|
| **Jurídico (clássico)** | Literal, lógico, histórico, teleológico, genético — valoriza o texto |
| **Tópico-problemático** | Viehweg — **problema** prevalece sobre norma |
| **Hermenêutico-concretizador** | Hesse — **texto** prevalece; pré-compreensão + caso concreto |
| **Integrativo (científico-espiritual)** | Smend — sistema de valores; Constituição dinâmica |
| **Normativo-estruturante (concretista)** | Müller — norma ≠ texto; contexto + aplicação |
| **Originalismo** | Sentido original (EUA/Scalia) — **pouco aplicável** no Brasil |
| **Lógica do razoável** | Recaséns Siches / Aarnio — ponderação e fundamentação pública |

### Princípios de interpretação (facultativos, sem valor normativo)

| Princípio | Função |
|-----------|--------|
| **Unidade** | Constituição como todo; antinomias só aparentes; sem norma originária inconstitucional |
| **Máxima efetividade** | Sentido que dê **maior eficácia social** (sobretudo direitos) |
| **Justeza / conformidade funcional** | Não subverter esquema organizacional do constituinte |
| **Concordância prática** | Harmonizar bens em conflito (colisão de direitos) |
| **Efeito integrador** | Favorecer integração política e social |
| **Força normativa** | Preferir soluções que atualizem e preservem eficácia da Carta |

**Pegadinha:** unidade = integração política → **errado** (isso é efeito integrador). Máxima efetividade = historicidade/atualização → **errado** (isso é força normativa).

### Interpretação conforme a Constituição

- Aplica-se a normas **infraconstitucionais**, não à CF em si.  
- Objetivo: **preservar validade** da lei.  
- Só para normas **plurissignificativas** — **não** para sentido **unívoco**.  
- Não pode deturpar sentido (**contra legem**).  
- Tipos: **com redução** do texto (suspensão de trecho) ou **sem redução** (concessiva/excludente).

## 3) princípios fundamentais (Título I — arts. 1º–4º)

### Regras × princípios (Alexy)

| | **Regra** | **Princípio** |
|---|-----------|---------------|
| Lógica | Tudo ou nada | Cumprimento **parcial** possível |
| Conflito | Uma **exclui** a outra | Conflito **aparente** — ponderação |

### Espécies (Canotilho)

- **Político-constitucionais:** decisões fundamentais do constituinte (Título I).  
- **Jurídico-constitucionais:** dispersos no texto (legalidade, juiz natural, devido processo).

**Pegadinha:** fundamentos (art. 1º), objetivos (art. 3º) e princípios internacionais (art. 4º) são **todos** “princípios fundamentais”.

### Art. 1º — fundamentos (mnemônico **SOCIDIVAPLU**)

| Inciso | Fundamento |
|--------|------------|
| I | Soberania (suprema/independente; soberania **popular** — § único) |
| II | Cidadania (status + participação política) |
| III | **Dignidade da pessoa humana** (valor-fonte; eficácia negativa e positiva; STF: união homoafetiva, células-tronco, DNA compulsório) |
| IV | Valores sociais do trabalho e da livre iniciativa (Estado capitalista + valor social do trabalho; art. 170) |
| V | Pluralismo político (exclui **discurso de ódio**) |

**Caput:** RFB = **federação** + **república** + **democracia** + **Estado democrático de direito**.

### Forma de Estado, governo e regime

| Conceito | Brasil |
|----------|--------|
| **Forma de Estado** | **Federação** (indissolúvel; sem direito de secessão; cláusula pétrea art. 60 §4º I) |
| **Forma de governo** | **República** (eletiva, representativa, transitória, responsável) |
| **Sistema de governo** | **Presidencialismo** (não confundir com forma de governo) |
| **Regime político** | **Democracia** semidireta/participativa |

**Federalismo brasileiro:** 3º grau (União, Estados, **Municípios**); centrífugo (unitário → estados); **cooperativo**; autonomia ≠ soberania (só a **RFB** é soberana; União tem autonomia, art. 21 I).

**Democracia semidireta:** voto + plebiscito + referendo + iniciativa popular + ação popular.  
- **Plebiscito:** **antes** da norma.  
- **Referendo:** **depois** da norma (ratificação).

**Princípios do EDD (José Afonso da Silva):** constitucionalidade; democrático; direitos fundamentais; justiça social; igualdade; separação de poderes + independência do juiz; legalidade; segurança jurídica.

### Art. 2º — separação de poderes

- Poder **único**; funções **distintas** (legislativa, executiva, judiciária).  
- Modelo **flexível** (funções atípicas: MP, MP por Judiciário/Legislativo, MPs/LDs).  
- **Independência** (sem hierarquia) + **harmonia** (colaboração).  
- Independência **não absoluta** → **freios e contrapesos** (origem EUA).  
- STF: freios estaduais só com **similaridade** com CF (ADI 1.905); vedado criar novos (ADI 3046).

### Art. 3º — objetivos (verbos: **Conga Erra Pro**)

| Inciso | Objetivo |
|--------|----------|
| I | **Construir** sociedade livre, justa e solidária |
| II | **Garantir** desenvolvimento **nacional** |
| III | **Erradicar** pobreza/marginalização; reduzir desigualdades |
| IV | **Promover** bem de todos, sem preconceitos (igualdade **material**; ações afirmativas) |

### Art. 4º — relações internacionais

| Inciso | Princípio |
|--------|-----------|
| I | Independência nacional |
| II | Prevalência dos direitos humanos |
| III | Autodeterminação dos povos |
| IV | Não intervenção |
| V | Igualdade entre Estados |
| VI | Defesa da paz |
| VII | Solução pacífica dos conflitos |
| VIII | Repúdio ao terrorismo e ao racismo |
| IX | Cooperação entre povos |
| X | Concessão de asilo político |

**§ único:** integração **América Latina** (não “América do Sul”) — é **objetivo**, não inciso.

**Pegadinha:** paz no art. **4º** VI, não no art. **1º**. Cooperação entre povos = art. **4º** IX, não fundamento.

## 4) teoria geral dos direitos fundamentais

### Conceitos

| Expressão | Sentido |
|-----------|---------|
| **Direitos do homem** | Naturais, jusnaturalistas, fora da positivação |
| **Direitos fundamentais** | Positivados na CF de um Estado |
| **Direitos humanos** | Tratados internacionais (Pacto ONU, CADH) |
| **Direitos** | Bens protegidos (vida, liberdade…) |
| **Garantias** | Instrumentos de proteção (HC, MS…) |

Proteção interna pode ser **superior** à dos tratados; tratado pode ter direito ainda não internalizado.

### Gerações / dimensões

| Geração | Valor-fonte | Natureza | Exemplos |
|---------|-------------|----------|----------|
| **1ª** | Liberdade | Negativas / defesa; obrigação de não fazer (mas pode exigir prestação) | Propriedade, locomoção, reunião, políticos |
| **2ª** | Igualdade | Positivas / bem-estar; muitas programáticas | Educação, saúde, trabalho |
| **3ª** | Solidariedade/fraternidade | Transindividuais (difusos/coletivos) | Meio ambiente, consumidor, desenvolvimento |

- Sequência Revolução Francesa: **Liberdade, Igualdade, Fraternidade**.  
- **4ª** (Bonavides): globalização (democracia, informação, pluralismo). Bobbio: engenharia genética. **5ª:** direito à paz.  
- Preferência doutrinária: **dimensões** (acumulativas, não substitutivas).

### Teoria dos status (Jellinek)

| Status | Relação com o Estado |
|--------|---------------------|
| **Passivo** | Sujeição a ordens/obrigações |
| **Negativo** | Liberdade sem interferência (expressão, ir e vir) |
| **Positivo** | Exigir prestações (educação gratuita) |
| **Ativo** | Direitos políticos (voto) |

Classificação trialista: defesa | prestações | participação.

### Características dos direitos fundamentais

| Característica | Ideia-chave |
|----------------|-------------|
| Universalidade | Núcleo comum; alguns só para grupos (trabalhadores) |
| Historicidade | Conquista progressiva; mutáveis |
| Indivisibilidade | Sistema harmônico |
| Inalienabilidade | Inegociáveis; sem conteúdo patrimonial |
| Imprescritibilidade | Sempre exigíveis |
| Irrenunciabilidade | Não dispor; pode autolimitar no caso (reality show) |
| **Relatividade** | **Nenhum absoluto** — harmonização proporcional |
| Complementaridade | Interpretação conjunta das dimensões |
| Concorrência | Exercício cumulativo |
| Efetividade | Dever estatal de concretizar |
| Proibição do retrocesso | Não enfraquecer conquistas (efeito cliquet) |

**Dimensões:** subjetiva (exigível do Estado) × objetiva (irradiam no ordenamento; eficácia irradiante).

**Cláusula pétrea:** art. 60 §4º IV — direitos do Título II (e além) insuscetíveis de **abolir** por EC.

### Título II — catálogo (arts. 5º–17)

| Categoria | Dispositivos |
|-----------|--------------|
| Individuais e coletivos | art. 5º |
| Sociais | arts. 6º–11 |
| Nacionalidade | arts. 12–13 |
| Políticos | arts. 14–16 |
| Partidos | art. 17 |

Rol **não exaustivo** — direitos **não catalogados** (art. 225 meio ambiente; art. 150 III b anterioridade).

## 5) limites e eficácia

### Teorias de limites

| Teoria | Ideia |
|--------|-------|
| **Interna (absoluta)** | Limites **imanentes** ao direito; núcleo fixo a priori |
| **Externa (relativa)** | Limites por colisão; **ponderação** e proporcionalidade no caso concreto |

**Brasil:** prevalece visão **externa** + **limites dos limites** (Schranken-Schranken): lei pode restringir, mas não violar **núcleo essencial** (proporcionalidade: adequação, necessidade, proporcionalidade em sentido estrito). Implícito na CF (Mendes/STF).

Restrições também em **estado de sítio/defesa** (arts. 136–137).

### Eficácia vertical × horizontal

| | Vertical | Horizontal |
|---|----------|------------|
| Relação | Indivíduo × **Estado** | **Particular × particular** |
| Brasil | Sim | **Sim** — tese **direta e imediata** (art. 5º §1º) |
| EUA | Sim | Não (só vertical) |

- Teoria indireta/mediate (cláusulas gerais do CC) → **incompatível** com art. 5º §1º.  
- **Eficácia diagonal:** relações assimétricas (trabalho patrão × empregado).

## pegadinhas

| Erro | Correção |
|------|----------|
| Desconstitucionalização no Brasil | **Não** adotada |
| Incompatível com nova CF = inconstitucional | **Revogado** (pré-constitucional) |
| Recepção exige lei expressa | Basta compatibilidade **material** |
| Compatibilidade formal necessária | **Irrelevante** para recepção |
| Lei em vacatio legis é recepcionada | **Não** — precisa estar em vigor |
| Só o Judiciário interpreta a CF | **Três Poderes** + sociedade (Häberle) |
| Não interpretativismo = texto literal | Isso é **interpretativismo** |
| Mutação altera texto | Altera **interpretação** apenas |
| Unidade = integração política | **Efeito integrador** |
| Máxima efetividade = atualização histórica | **Força normativa** |
| Interpretação conforme em norma unívoca | **Vedada** |
| Paz = fundamento art. 1º | **Art. 4º** VI |
| Desenvolvimento internacional (art. 3º) | **Nacional** |
| Sociedade soberana (art. 3º) | **Livre**, justa e solidária |
| Presidencialismo = forma de governo | **Sistema** de governo; forma = **República** |
| Democracia só indireta | **Semidireta** (plebiscito, referendo, IP, AP) |
| Plebiscito após a norma | **Antes**; referendo = **depois** |
| Autonomia = soberania | Só **RFB** é soberana |
| Direito absoluto | **Relatividade** — todos limitáveis |
| Meio ambiente = 2ª geração | **3ª** geração |
| Dimensão subjetiva = irradiante | **Objetiva** irradia no ordenamento |
| Universalidade = todos titulam todos | Alguns direitos são **grupo-específicos** |
