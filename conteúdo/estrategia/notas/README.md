# Notas destiladas — Estratégia Concursos

Resumos reescritos a partir dos PDFs (conversa ou `estrategia/`). **Não** substituem lei seca nem provas IDECAN.

## Onde salvar

`notas/{disciplina}/{topico-ou-recorte}.md` — ex.: `notas/legislacao-transito/art-252-celular.md`

## Seções por arquivo

| Seção | Conteúdo |
|-------|----------|
| `recorte_edital` | Item do edital/retificação coberto |
| `ordem_estudo` | Pré-requisitos e sequência sugerida |
| `pegadinhas` | Erros frequentes em concurso (reescritos) |
| `fundamento_oficial` | Links para arquivos em `conteúdo/` (CTB, resoluções…) |
| `nao_confundir_com` | Artigos/temas vizinhos que confundem |

## Regras

- Paráfrase própria — sem trechos longos da apostila
- Todo artigo/número/prazo: validar em `conteúdo/` antes de usar em questão ou aula
- Índice geral: `conteúdo/FONTES.md` § Material Estratégia
