## pegadinhas

### Art. 161 — conceito

| Regra | Conteúdo |
|-------|----------|
| **161** | Infração = inobservância de preceito do CTB ou legislação complementar → penalidades e medidas do **próprio capítulo** + punições do **Cap. XIX** (crimes de trânsito) |
| **161 — pegadinha** | Penalidade exige **processo** (não é autoexecutória); medida administrativa pode ser **imediata** no flagrante |

### Multiplicadores de multa (gravíssimas) — memorizar

| Fator | Exemplos (apostila / CTB) |
|-------|---------------------------|
| **×20** | Art. **253-A** — usar veículo para interromper/restringir/perturbar circulação **sem autorização**; organizadores: **×60** (§1º); reincidência 12 meses: **dobro** (§2º) |
| **×10** | Arts. **165**, **165-A** (álcool / recusa ao teste); **173** (racha); **174** (competição não autorizada); **175** (manobra perigosa); **191** (forçar passagem entre veículos) |
| **×5** | Arts. **165-B**, **165-C** (toxicológico); **176** (sinistro com vítima — omissões); **202–203** (ultrapassagem proibida); **230 XX** (transporte escolar sem autorização); **246** (obstáculo não sinalizado — até ×5 a critério da autoridade) |
| **×3** | Art. **162** I e II (sem CNH / CNH cassada ou suspensa); **163–164** (entregar/permitir dirigir nessas condições); **193** (transitar em calçadas, ciclovia, acostamento etc.); **218 III** (velocidade **>50%** acima do máximo) |
| **×2** | Art. **162** III (categoria errada); **163–164** (categoria errada); reincidência **165** e **165-A** (12 meses) |

**Valores base em R$:** conferir vigência (apostila cita 293,47 / 195,23 / 130,16 / 88,38).

### Arts. 162–165-D — dirigir, habilitação, álcool e toxicológico

| Art. | Gravidade / destaque |
|------|----------------------|
| **162 I** | Sem CNH/PP/ACC → gravíssima; multa **×3**; retenção até condutor habilitado |
| **162 II** | CNH cassada ou suspensa → gravíssima; multa **×3**; recolhimento CNH + retenção |
| **162 III** | Categoria diferente → gravíssima; multa **×2**; retenção |
| **162 V** | CNH vencida há **>30 dias** → gravíssima; multa simples; retenção |
| **162 VI** | Sem lentes/aparelho/prótese/adaptações impostas → gravíssima |
| **162 VII** | Sem curso especializado obrigatório → gravíssima |
| **163–164** | Entregar/permitir que outrem dirija sem habilitação, categoria errada, suspensa/cassada etc. → espelha **162** (×3 ou ×2) |
| **165** | Álcool/psicoativo → gravíssima; multa **×10** + suspensão **12 meses**; recolhimento CNH + retenção/remoção (art. 270 §4º); reincidência 12m: multa **dobro** |
| **165-A** | Recusa a teste/exame/perícia (art. 277) → **mesmas consequências do 165** |
| **165-B** | Dirigir sem exame toxicológico (148-A) → gravíssima; multa **×5**; reincidência 12m: **×10** + suspensão; configura-se após **30º dia** do vencimento do prazo (§ único + 148-A §2º) |
| **165-C** | Dirigir com toxicológico **positivo** → mesmas penas do **165-B** |
| **165-D** | Deixar de realizar toxicológico no prazo do **148-A §2º** → gravíssima (penas na lei — cruzar HTML) |
| **166** | Confiar/entregar direção a habilitado **incapaz** física/psiquicamente → gravíssima (diferente de **163**, que trata **falta** de habilitação) |

### Arts. 167–178 — conduta, sinistro e socorro

| Art. | Gravidade / penalidade |
|------|------------------------|
| **167** | Sem cinto (condutor ou passageiro) → **grave** |
| **168** | Criança sem dispositivo de retenção → **gravíssima** |
| **169** | Sem atenção/cuidados → **leve** (não confundir com **170**) |
| **170** | Ameaçar pedestre/veículo → **gravíssima** + **suspensão** |
| **171** | Arremessar água/detritos → **média** (não leve) |
| **172** | Atirar/abandonar objetos na via → **média** |
| **173–175** | Racha, competição não autorizada, manobra perigosa → gravíssima; multa **×10**; suspensão |
| **176** | Sinistro **com vítima** — omitir socorro, sinalização, preservar local, remover se determinado, identificar-se → gravíssima; multa **×5** + **suspensão**; recolhimento CNH |
| **177** | Não prestar socorro quando **solicitado** pela autoridade (não envolvido) → **grave** |
| **178** | Sinistro **sem vítima** — não remover veículo quando necessário → **média** |

### Arts. 181 × 182 — estacionar × parar (tabela-chave)

Mesmo local pode ter gravidade **diferente** conforme o verbo:

| Situação | **Estacionar (181)** | **Parar (182)** |
|----------|----------------------|-----------------|
| Pista de rolamento (rodovia/via rápida/com acostamento) | **Gravíssima** | **Grave** |
| Afastado do meio-fio **>1 m** | **Grave** | **Média** |
| Afastado **50 cm a 1 m** | **Leve** | **Leve** |
| Cruzamento prejudicando circulação | **Grave** | **Média** |
| Pontes, viadutos, túneis | **Grave** | **Média** |
| Passeio, faixa de pedestre, ciclovia, ilhas, canteiros etc. | **Grave** (+ remoção) | Ver incisos (muitos **leve/média**) |
| Vaga PcD/idoso sem credencial | **Gravíssima** + remoção | — |
| Sobre ciclofaixa/ciclovia (**parar**) | — | **Grave** (182 XI) |

| **183** | Parar sobre faixa de pedestres na mudança do sinal → **média** |

### Arts. 218–220 — velocidade

| Faixa (art. 218) | Gravidade | Penalidade extra |
|------------------|-----------|------------------|
| Até **20%** acima do máximo | **Média** | Multa |
| **>20% até 50%** | **Grave** | Multa |
| **>50%** | **Gravíssima** | Multa **×3** + **suspensão** |
| **219** | Abaixo da **metade** da mínima permitida → **média** |
| **220** | Não reduzir velocidade (escola, hospital, ciclista, aglomeração etc.) → **gravíssima** |

### Arts. 230–236 — veículo, documentos e carga

| Art. | Destaque |
|------|----------|

| **231** | Lotação excedente; transporte remunerado irregular → **gravíssima** |

## nao_confundir_com

| Confusão | Correção |
|----------|----------|
| Estacionar = parar | **181** (estacionar) em regra é **mais grave** que **182** (parar) no mesmo local |
| 169 = 170 | **169** leve (desatenção); **170** gravíssima + suspensão (ameaça) |
| 176 = 177 | **176** envolvido em sinistro **com vítima** (gravíssima ×5); **177** não envolvido, só se autoridade pedir (grave) |
| 165 = 165-A | Penas **idênticas** (álcool = recusa ao teste) |
| 165-B = 165-C | Mesmo bloco toxicológico (ausência vs positivo); prazo após vencimento: **30 dias** (165-B § único) |
| 252 VI = celular na mão | **Fones** = média (VI); **segurar/manusear** = gravíssima (§ único do V) |
| 253 = 253-A | **253** bloqueio simples (apreensão); **253-A** ato **deliberado** de perturbar circulação (×20 + suspensão) |
| 210 = 253-A | **210** transpor bloqueio **policial**; **253-A** bloqueio por manifestação etc. sem autorização |
| 232 = 162 I | Sem **documento no porte** = leve; sem **habilitação** = gravíssima ×3 |
| 184 II = 184 III | Faixa esquerda exclusiva = **grave**; faixa de **ônibus** = **gravíssima** |
| Medida × penalidade | Ver `indice-ctb-aula07.md` — retenção no flagrante ≠ multa sem processo |
| Infração × crime | Cap. XV (infrações) vs Cap. XIX (crimes) — art. 161 remete aos dois |
